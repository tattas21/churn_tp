```
/ml-churn              → guía de modelado y evaluación

/eda-churn hipótesis   → exploración de hipótesis específicas
/eda-churn outliers    → detección y tratamiento

/negocio-ecommerce     → contexto de negocio del churn
/negocio-ecommerce RFM → segmentación RFM
/negocio-ecommerce impacto → estimación del valor económico del modelo
```
